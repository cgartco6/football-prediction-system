# Football Prediction System

Advanced self-learning soccer predictor (1X2 + BTTS) with deep factors (injuries, weather, H2H, coach, key players, etc.).

## Alternative Free APIs (2026)
- football-data.org (default)
- API-Football / API-Sports (recommended for injuries + stats)
- Sportmonks, OpenLigaDB

## Quick Start
1. `python -m venv venv`
2. Activate venv & `pip install -r requirements.txt`
3. Copy `.env.example` to `.env` and add your API key
4. Run `start.bat` (Win), `start.ps1`, or `./start.sh`
